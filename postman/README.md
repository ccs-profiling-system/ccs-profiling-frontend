# Postman Collection

This folder contains Postman collections for testing the CCS Profiling API.

## 📌 Usage

1. Import the collection from `collections/`
2. Import environment from `environments/`
3. Set your local `base_url`
4. Login to get token
5. Set token in environment

## ⚠️ Notes

- Do not commit real tokens
- Use placeholders only
- Update environment variables as needed
